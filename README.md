# 🪔 Diwali Sales Analysis using Python

## 📌 Project Overview

This project focuses on analyzing Diwali sales data to understand customer purchasing behavior and identify key business insights. Using Python and data analysis libraries, the dataset is cleaned, explored, and visualized to uncover trends that can help improve sales strategies.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze customer demographics and buying patterns
* Identify top-performing products and categories
* Extract actionable business insights
* Visualize trends using graphs and charts

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset

The dataset contains information about:

* Customer details (age, gender, occupation, city)
* Product categories
* Purchase amounts

---

## 🔍 Key Analysis Performed

* Data cleaning (handling missing values, formatting)
* Exploratory Data Analysis (EDA)
* Gender-wise and age-group analysis
* State-wise sales performance
* Product category analysis
* Top customers and high-revenue segments

---

## 📊 Insights & Findings

* Identified which gender contributes more to overall sales
* Found the most active age group of buyers
* Determined top-performing states during Diwali sales
* Highlighted best-selling product categories
* Discovered high-value customer segments

---

## 📈 Visualizations

* Bar charts for sales comparison
* Count plots for customer distribution
* Heatmaps for correlation analysis
* Category-wise and state-wise analysis graphs

---

## 🚀 How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/diwali-sales-analysis.git
   ```

2. Navigate to the project folder:

   ```bash
   cd diwali-sales-analysis
   ```

3. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

4. Open the notebook:

   ```bash
   jupyter notebook
   ```
